## 具体的な手順

### [Nodejsの公式サイト](https://nodejs.org/ja/download/)からインストーラーをダウンロード

### プロジェクトを以下のコマンドで作る
```
npx create-next-app@latest my-app --typescript 
```

ここで，　my-appがプロジェクト名であり，--typescriptで使用言語が指定できる．今回は，

```
npx create-next-app@latest studynext --javascripts
```
として，プロジェクトを作成した．
